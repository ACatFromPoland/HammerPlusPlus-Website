Hammer++ supports different branches of Source engine.

Alternative

There is basic temporary Garry's Mod support if you install the TF2 or SDK 2013 Multiplayer version.
If "garrysmod" is detected in the game configuration mod path,
Hammer++ will automatically attempt to mount any games from your Garry's Mod.
Does not work with all mounted games and might have some bugs.
